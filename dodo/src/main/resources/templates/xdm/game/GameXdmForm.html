파일 선택
파일의 형식과 크기를 확인해 주세요.
게임명을 입력해 주세요. 한글/영문/숫자만 허용
게임 카테고리를 선택해 주세요.
난이도를 숫자(0 ~ 5 사이의 실수, 소수점 두 자리)로 입력해 주세요.
최저 사용 연령을 숫자(0보다 큰 정수)로 입력해 주세요.
최대 시간보다 작거나 같고, 0보다 큰 정수(분 단위)로 입력해 주세요.
최소 소요 시간보다 크거나 같고, 0보다 큰 정수(분 단위)로 입력해 주세요.
최대 인원보다 작거나 같고, 0보다 큰 정수로 입력해 주세요.
최소 인원보다 크거나 같고, 0보다 큰 정수로 입력해 주세요.
최적인원을 숫자(0보다 큰 정수, 최대인원 이하)로 입력해 주세요.
추천인원을 숫자(0보다 큰 정수, 최대인원 이하)로 입력해 주세요.
공식판매사를 선택해 주세요.
공식 판매 주소를 URL 형식에 맞게 입력해 주세요.
보드라이프 주소를 URL 형식에 맞게 입력해 주세요.
Geek 주소를 URL 형식에 맞게 입력해 주세요.
디자이너 이름을 입력해 주세요. (여러명일 경우 줄바꿈으로 구분)
아트웍 작가 이름을 입력해 주세요. (여러명일 경우 줄바꿈으로 구분)
출판사 이름을 입력해 주세요. (여러곳일 경우 줄바꿈으로 구분)
